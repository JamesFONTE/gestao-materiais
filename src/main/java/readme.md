# Sistema de Gestão de Materiais

Aplicação web desenvolvida como projeto acadêmico para praticar desenvolvimento backend com **Java e Spring Boot**, utilizando **Thymeleaf** para renderização das páginas.

O sistema realiza o controle básico de materiais utilizados em uma obra.

---

## Funcionalidades

- Cadastro de materiais
- Listagem de materiais
- Edição de materiais
- Exclusão de materiais

---

## Tecnologias Utilizadas

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- HTML
- CSS
- Maven

---

## Como Executar o Projeto

### Pré-requisitos
- Java 11 ou superior
- Maven

### Executando
```bash
mvn spring-boot:run
http://localhost:8080/materiais
